При загрузке страницы запросить у пользователя ввести имя (функция prompt()). С помощью метода document.write вывести на экран строку «Привет, введенное имя !».При помощи функции prompt() попросить пользователя ввести 2 числа. Выполнить произведение чисел и вывести результат на экран с помощью метода document.write.
