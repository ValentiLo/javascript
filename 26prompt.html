prompt(). Синтаксис: [window.]prompt(сообщение,значение_поля_ввода). С помощью окна ввода, вызываемого методом prompt, пользователь может ввести данные, которые будут использоваться далее в сценарии. 26/. prompt("Как Вас зовут?", "Иван"); Важно: Метод prompt принимает два аргумента: первый выводится в качестве простой строки в модальном окне; второй — значение по умолчанию в текстовом поле для ввода. Оба аргумента заключаются в кавычки.
